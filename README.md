# 鉄輪式戦車レース 演出アプリ MVP

Discord画面共有を想定した、戦車レース1レース進行用のローカルWebアプリです。

## 起動方法

依存ライブラリをインストールしてから、Flaskアプリを起動します。

```bash
pip install flask
python app.py
```

起動後、ブラウザで以下を開きます。

- GM管理画面: `http://127.0.0.1:5000/control`
- 観客表示画面: `http://127.0.0.1:5000/display`

## 画面

### `/control`

GM用の管理画面です。

- ランク選択 D/C/B/A/S
- 番組選択 標準/高速/技巧/重装/射撃/荒れ場
- seed指定
- 1レース自動生成
- イベントの前後移動
- 現在イベント、出走表、イベント一覧の確認

### `/display`

プレイヤー・観客に見せる表示画面です。

- 三列エリアの現在配置
- 戦車カード
- 現在順位
- 実況文
- ラウンド名とseed

Discordで画面共有する場合は、GM管理画面ではなく `/display` を共有してください。

## Ver.0.4 追加仕様

`sim_engine.py` は現行Ver.0.4相当の簡易シミュレーターです。

Ver.0.4では、レース開始前に配置決めフェーズを行います。

- 前列狙い: `2d6 + 機動力 + 操縦性`、目標値9。
- 前列狙い成功: 前列配置、駆動力-1。
- 前列狙い失敗: 中列配置、駆動力-1、安定値-1。
- 出目2: 中列配置、駆動力-1、安定値-2。
- 出目11以上: 自動成功かつ大成功。前列配置、駆動力消費なし。
- 中列配置: 補正なし。
- 後列配置: 駆動力+1。

## 高速戦の構成変更

通常の高速戦は、Ver.0.4で以下の構成になりました。

- 逃げ2
- 追込2
- 万能2

旧来の逃げ3構成は、必要なら別番組として追加する想定です。
