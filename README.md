# 鉄輪式戦車レース 演出アプリ MVP

Discord画面共有を想定した、戦車レース1レース進行用のローカルWebアプリです。

## 起動方法

```bash
pip install flask
python app.py
```

起動後、ブラウザで以下を開きます。

- GM管理画面: http://127.0.0.1:5000/control
- 観客表示画面: http://127.0.0.1:5000/display

Discordでは観客表示画面だけを共有してください。

## できること

- ランク選択 D/C/B/A/S
- 番組選択 標準/高速/技巧/重装/射撃/荒れ場
- seed指定
- 1レース自動生成
- GM画面からイベントを1つずつ進行
- 観客画面で三列エリア、戦車カード、実況文を表示

## 注意

- 現時点ではMVPです。
- PC手動操作、賭け金管理、戦車編集、詳細オッズ表示は未実装です。
- `sim_engine.py` は現行Ver.0.4相当の簡易シミュレーターです。
- Ver.0.4では、レース開始前に配置決めフェーズを行います。
  - 前列狙い: `2d6 + 機動力 + 操縦性`、目標値9。成功で前列、失敗で中列。失敗時は駆動力-1・安定値-1。
  - 中列配置: 補正なし。
  - 後列配置: 駆動力+1。
- 通常の高速戦は、逃げ2・追込2・万能2の構成です。
